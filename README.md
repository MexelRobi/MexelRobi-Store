# MexelRobi-Store (AltStore Source)

Welcome to the official **MexelRobi-Store**, a custom source for [AltStore](https://altstore.io/). This repository provides useful tools for your iOS device that can be easily installed and kept up-to-date via AltStore.

## 🚀 How to Install

To add this source to your AltStore, follow these simple steps:

1. Copy the following URL:
   `https://mexelrobi.github.io/MexelRobi-Store/altstore.json`
2. Open **AltStore** on your iOS device.
3. Navigate to the **"Sources"** tab.
4. Tap the **"+"** icon in the top left corner.
5. Paste the URL and tap **"Add"**.

## 📱 Featured Apps

Currently, this store provides the following applications:

### 1. Respringer
A fast and reliable tool to perform a **Respring** on your device. Perfect for users who need to reload the SpringBoard quickly without a full reboot.

### 2. BreakBox
A utility tool that allows you to launch any installed application directly using its unique Bundle ID.
